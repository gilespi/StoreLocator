# StoreLocator
Responsive Leaflet map store locator demo project.

The application automatically determines the coordinates by entering the full address.

link: https://storelocator.cyclic.app/?#

# Install

npm run dev

# Start demo site

localhost:5000

# Plugins I used in this project

*axios
*cors
*dotenv
*express
*mongoose
*node-geocoder
*nodemon
